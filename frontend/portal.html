Antivirus False Positive Flag

Because the 3RD AI KAVACH background agent (`3rd-AI-Agent.exe` / `kockroch.exe`) executes system commands (Sleep/Lock/Logs), Windows Defender flags it as unsafe. This is a false positive.

  • Open Windows Security settings and navigate to Virus & threat protection.
  • Under Virus & threat protection settings, click on Manage settings.
  • Scroll down to Exclusions, and click on Add or remove exclusions.
  • Click Add an exclusion, select Folder, and whitelist your target directory: `C:\Users\Public\CyberGuardian_Data`.
  • Alternatively, you can automatically add the folder path exception by running this PowerShell command:
Add-MpPreference -ExclusionPath "C:\Users\Public\CyberGuardian_Data"
